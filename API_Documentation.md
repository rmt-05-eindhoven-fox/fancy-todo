# TODO

**Add new ToDo**
----
  Create new ToDo Object

* **URL**

  `/todos`

* **Method:**
  
  `POST` 

* **URL Params**

  None

* **Data Params**
```json
  {
    "title" : "Work on API Docs",
    "description" : "API Docs for Fancy ToDos",
    "status" : "Not Done",
    "due_date" : "2020-10-30"
  }
```

* **Success Response:**

  * **Code:** 201 CREATED <br />
  **Content:** 
  ```json
    {
      "id" : 1,
      "title" : "Work on API Docs",
      "description" : "API Docs for Fancy ToDos",
      "due_date" : "2020-10-30"
    }
  ```
 
* **Error Response:**

  * **Code:** 400 BAD REQUEST <br />
  <!-- **Content:** `{ error : "Log in" }` -->
