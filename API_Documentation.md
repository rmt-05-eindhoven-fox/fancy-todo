**Add new ToDo**
----
  Create new ToDo Object

* **URL**

  /todos

* **Method:**
  
    `POST` 

*  **URL Params**

   None

* **Data Params**
```
    {
        "title" : "Work on API Docs",
        "description" : "API Docs for Fancy ToDos",
        "status" : "Not Done",
        "due_date" : "2020-10-30"
    }
```

* **Success Response:**

  * **Code:** 201 CREATED <br />
**Content:** 
```
    {
        "id" : 1,
        "title" : "Work on API Docs",
        "description" : "API Docs for Fancy ToDos",
        "due_date" : "2020-10-30"
    }
```
 
* **Error Response:**

  * **Code:** 400 BAD REQUEST <br />
    <!-- **Content:** `{ error : "Log in" }` -->

**Get one ToDo**
----
  Return a ToDo object based on given URL params

* **URL**

  /todos/:id

* **Method:**
  
    `GET` 

*  **URL Params**

    id refers to id of ToDo object

    **Required:**
 
   `id=[integer]`

* **Data Params**

   None


* **Success Response:**

  * **Code:** 200 OK <br />
**Content:** 
```    
    {
        "id" : 1,
        "title" : "Work on API Docs",
        "description" : "API Docs for Fancy ToDos",
        "due_date" : "2020-10-30"
    }

```
 
* **Error Response:**

  * **Code:** 404 NOT FOUND <br />
    <!-- **Content:** `{ error : "Log in" }` -->
